# Data Structures Study Repository

This repository is dedicated to studying and implementing fundamental data structures from scratch, with a focus on understanding their internal mechanics, performance characteristics, and real-world use cases.

The goal is to build a solid foundation in computer science by exploring how data structures work under the hood, rather than relying only on built-in abstractions.

## Topics Covered

- Arrays & Linked Lists
- Stacks & Queues
- [Trees](./trees/) (Binary Trees, BST, etc.)
- Segment Trees
- Graphs
- Hash Tables
- Heaps & Priority Queues
